Print x Twist

PRINT x TWIST is a concept project of a lamp intended to use new industrialization technologies for a greater personnalization and a lower carbon footprint.

Date

2021

My role

Industrial design
Product design

Duration

1 week

Team size

Only me

The use of 3D-printing allows the users to chose the model and color in a much larger range.

Receiving the package

The first step from the user is receiving the package after ordering the base from the website

Unpacking

The user can now unpack the small package containing the base, LED strip and battery

Accessing the 3D files

The user can download the desired shape thanks to the QR code found in the packaging

3D printing the desired shape

Once done, he can print the shape with his own 3D printer, or in a FabLab

Assembling the lamp

Once the handle and shade printed in the desired colors, the user can assemble them with the base as simply as a twist!

Turning it on!

He can now turn the lamp on

This projects was made in order to reduce the environmental footprint of industrial objects by reducing the size of packages thanks to the local production of certain parts of the lamp thanks to 3D printing.

Electronics

Included in the packaging

The box contains the base and the different electronic parts. It includes a LED strip, a USB-C module, a rechargeable battery and a wireless charging coil.

Plastic parts

To be 3D-printed

The user can 3D-print the diffuser and the handle at his home if he owns a 3D-printer, or go to the nearest FabLab.

Variety of shapes

Using 3D printing means there could be a big amount of shapes to choose from, not only three. The user could also personnalize the shape to find the one that truly fits him.
